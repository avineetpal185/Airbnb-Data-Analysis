# Airbnb-Data-Analysis
Exploratory Data Analysis (EDA) on Airbnb listing dataset to identify pricing patterns, listing distribution, and availability trends using Python.

# Airbnb Data Analysis Project

## Project Overview
This project analyzes Airbnb listing data to understand price distribution, location trends, and availability.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
1. Data cleaning
2. Price distribution analysis
3. Location-based listing analysis
4. Availability trends
5. Visualization of listing patterns

## Key Insights
- Most listings price range between 800–1100
- Some areas have significantly higher pricing
- Availability varies by location

## Dataset
Airbnb listing dataset
